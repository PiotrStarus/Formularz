Tranzycje

Co to tranzycje CSS?

Tranzycje CSS pozwalaja na płynną zmianę wartości właściwości, przez dany czas trwania.


Jak używać tranzycji CSS?

Do stworzenia efektu tranzycji, musimy podać dwie rzeczy:

  • Właściwość CSS do której chcesz dodać efekt
  • Czas trwania efektu

    • Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transition: width 2s;

    }


    div:hover {

    width: 300px;

    }



    Określanie krzywej prędkości przejścia

    Właściwość "transition-timing-function" określa krzywą prędkości efektu przejścia.

    Może mieć ona następujące wartości:

  • ease -> efekt przejścia typu wolno-szybko-wolno (defaultowe ustawienie)
  • linear -> efekt przejścia z tą samą prędkością od początku do końca
  • ease-in -> efekt przejścia z wolnym startem
  • ease-out -> efekt przejścia z wolnym końcem
  • ease-in-out -> efekt przejścia z wolnym startem i końcem
  • cubic-bezier(n,n,n,n) -> pozwala zdefiniować swoje własne wartości przejścia

    • Przykład

    EASE
    LINEAR
    EASE-IN
    EASE-OUT
    EASE-IN-OUT


    Opóźnienie efektu tranzycji

    Właściwość "transition-delay" określa opóźnienie (w sekundach) dla efektu tranzycji.

    Następujący przykład ma 1 sekundę opóźnienia przed uruchomieniem:

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transition: width 2s;

    transition-delay: 1s;

    }


    div:hover {

    width: 300px;

    }











    Transformacje

    Co to transformacje CSS?

    Transformacje CSS pozwalaja na poruszanie, obracanie, skalowanie i przekrzywianie elementów.

    Za pomocą właściwości "transform" możemy używać następujących metod transformacji:

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()
  • rotateX()
  • rotateY()
  • rotateZ()


    • Translate()

    translate() przesuwa element z jego aktualnego miejsca według podanych parametrów.

    Następujący przykład przesuwa element div o 50 pixeli w prawo i 50 pixeli w dół.

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: translate(50px, 50px);

    }



    Rotate()

    rotate() obraca element według wskazówek zegara lub odwrotnie, według podanych stopni.

    Następujący przykład obraca element div o 20 stopni według wskazówek zegara.

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: rotate(20deg);

    }



    Scale()

    scale() zwiększa lub zmniejsza rozmiar elementu, według podanych parametrów dla szerokości i wysokości.

    Następujący przykład zmniejsza element div tak aby byl 2 razy węższy i 2 razy niższy niż oryginał.

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: scale(0.5, 0.5);

    }



    Skew()

    skew() przekrzywia element wzdłuż osi X i Y według podanych stopni.

    Następujący przykład przekrzywia element div o 20 stopni wzdłuż osi X i o 10 stopni wzdłuż osi Y:

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: skew(20deg, 10deg);

    }



    Matrix()

    matrix() łączy wszystkie podane wyżej metody w jedno.

    Parametry podajemy następująco: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: matrix(0.5, -0.3, 1.3, 0.7, 15, 10);

    }



    RotateX()

    rotateX() obraca element wzdłuż osi X według podanych stopni.

    Następujący przykład obraca element div o 40 stopni wzdłuż osi X

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: rotateX(40deg);

    }



    RotateY()

    rotateY() obraca element wzdłuż osi Y według podanych stopni.

    Następujący przykład obraca element div o 40 stopni wzdłuż osi Y

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: rotateY(40deg);

    }



    RotateZ()

    rotateZ() obraca element wzdłuż osi Z według podanych stopni.

    Następujący przykład obraca element div o 40 stopni wzdłuż osi Z

    Przykład

    div {

    width: 100px;

    height: 100px;

    background: #1C2841;

    transform: rotateZ(40deg);

    }











    Animacje

    Co to animacje CSS?

    Animacje CSS pozwalaja na stopniową zmianę stylu elementu z jednego na inny.

    Możesz zmieniać tyle właściwości CSS ile tylko potrzebujesz.

    Aby użyć animację CSS, musisz najpierw określić kilka klatek animacji.

    Klatki zawierają style, jakie element będzie miał w określonych momentach.


    Jak używać animacje CSS?

    Do stworzenia efektu animacji musimy dodać klatki i style do @keyframes.

    Animację dodajemy do elementu przez właściwość "animation-name".

    Ustawiamy długości czasu przez jaki wykonywać ma się funkcja we właściwości "animation-duration".

    W elemencie możemy ustawić ilość powtórzeń animacji przez "animation-iteration-count (domyslne = 1).


    W następującym przykładzie element div świeci się w kolorach tęczy

    oraz przekrzywia się od 0 do 20 do 0 stopni co klatkę według osi X w czasie 3 sekund:

    Przykład

    /* Kod Animacji */

    @keyframes przyklad {

    0% {background-color: red; skewX(0deg);}

    25% {background-color: yellow; skewX(5deg);}

    50% {background-color: blue; skewX(10deg);}

    75% {background-color: green; skewX(5deg);}

    100% {background-color: red; skewX(0deg);}

    }

    /* Element do którego dodajemy animacje */

    div {

    width: 100px;

    height: 100px;

    background: red;

    animation-name: przyklad;

    animation-duration: 3s;

    animation-iteration-count: infinite;

    }



    Opóźnienie animacji

    Właściwość "animation-delay" określa opóźnienie przed startem animacji.

    Może mieć ona wartości dodatnie jak i ujemne

    Wartość ujemna sprawia, że animacja zaczyna się tak jakby trwała już daną liczbę sekund


    Animacja poniżej włącza się dopiero 10 sekund od odświeżenia strony

    Przykład

    /* Kod Animacji */

    @keyframes przyklad {

    0% {background-color: red; skewX(0deg);}

    25% {background-color: yellow; skewX(5deg);}

    50% {background-color: blue; skewX(10deg);}

    75% {background-color: green; skewX(5deg);}

    100% {background-color: red; skewX(0deg);}

    }

    /* Element do którego dodajemy animacje */

    div {

    width: 100px;

    height: 100px;

    background: red;

    animation-name: przyklad;

    animation-duration: 3s;

    animation-iteration-count: infinite;

    animation-delay: 10s;

    }



    Odtwarzanie animacji w odwrotnym kierunku

    Właściwość "animation-direction" określa w jakim kierunku animacja powinna zostać odtworzona.

    Właściwość "animation-direction" może mieć następujące wartości:

  • normal -> Animacja jest odtwarzana normalnie (do przodu). To ustawienie jest domyślne.
  • reverse -> Animacja odtwarzana jest w przeciwną stronę (od tyłu).
  • alternate -> Animacja odtwarzana jest najpierw do przodu, a następnie do tyłu.
  • alternate-reverse -> Odwrotność alternate

    • Krzywe prędkości

    Właściwość "animation-timing-function" określa krzywą prędkości animacji.

    Właściwość "animation-timing-function" może mieć następujące wartości:

  • ease -> efekt przejścia typu wolno-szybko-wolno (defaultowe ustawienie)
  • linear -> efekt przejścia z tą samą prędkością od początku do końca
  • ease-in -> efekt przejścia z wolnym startem
  • ease-out -> efekt przejścia z wolnym końcem
  • ease-in-out -> efekt przejścia z wolnym startem i końcem
  • cubic-bezier(n,n,n,n) -> pozwala zdefiniować swoje własne wartości przejścia

    • Poniższa animacja pokazuje zastosowanie krzywych prędkości jak i odtwarzanie w cyklach przemiennych:

    Przykład

    /* Kod Animacji */

    @keyframes move {

    from {margin-left: 3%;}

    to {margin-left: 53%;}

    }

    img {

    width: 50px;

    height: 50px;

    animation-name: move;

    animation-duration: 5s;

    animation-iteration-count: infinite;

    animation-direction: alternate;

    animation-timing-function: linear;

    animation-delay: 500ms;

    }
